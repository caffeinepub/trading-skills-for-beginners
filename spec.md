# Specification

## Summary
**Goal:** Build a professional, single-page article landing page titled "Trading Skills for Beginners" with a dark charcoal and gold editorial theme.

**Planned changes:**
- Create a hero section with a full-width banner image, the article title, and a subtitle
- Add a sticky table of contents for navigation between sections
- Implement three content sections: "Basics of Trading" (assets, markets, buy/sell mechanics), "Background of Trading" (history and evolution of markets), and "How to Start Trading" (account setup, broker selection, risk management, first trades)
- Apply a dark charcoal and gold color palette with a serif headline font and sans-serif body font
- Set appropriate meta title and description for the article
- Use semantic HTML elements (h1, h2, h3, article, section) throughout

**User-visible outcome:** Users can read a well-structured, visually polished beginner trading article with clear sections, easy navigation, and a professional financial editorial look.
